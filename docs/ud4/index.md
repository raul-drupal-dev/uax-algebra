# Unidad 4 — Diagonalización (Índice)

En esta unidad trabajamos la diagonalización de matrices y la forma canónica de Jordan. La unidad está pensada para cubrir todos los pasos necesarios: definiciones, criterios, algoritmos paso a paso y muchos ejercicios resueltos para practicar "las trampas" más comunes.

Páginas de la unidad:

- Definiciones y conceptos básicos — `definiciones.md`
- Criterios y comprobaciones para diagonalizar — `criterios-diagonalizacion.md`
- Algoritmo paso a paso y ejemplo detallado — `diagonalizacion-paso-a-paso.md`
- Forma canónica de Jordan — `forma-jordan.md`
- Ejercicios resueltos (varios tipos) — `ejercicios-resueltos.md`
- Ejercicios propuestos para practicar — `ejercicios-propuestos.md`
- Resumen de la UD4 — `resumen.md`

> Consejo: sigue las páginas en el orden propuesto: primero definiciones, luego criterios, después el algoritmo y finalmente practica con los ejercicios.
